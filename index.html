Html

HTML (HyperText Markup Language) is the most basic building block of the web. It is a markup language used for structuring and presenting content on the web. HTML consists of a series of elements, which you can use to create and format web pages. HTML elements tell the browser how to display the content. HTML is a very simple markup language, and you can learn it in just a few days.

Css

CSS stands for Cascading Style Sheets. CSS is a language used to style and control the look and layout of webpages. It is used to define the font, color, size, spacing, and other design elements of HTML elements. It is also used to set the background color, background images, and other design elements of an entire website.

Javascript

JavaScript is a programming language used to make webpages interactive. It is the most popular scripting language on the web and is used to create dynamic websites, games, and applications. JavaScript can be used to create animations, change page content, and to add various types of interactivity to a webpage.
